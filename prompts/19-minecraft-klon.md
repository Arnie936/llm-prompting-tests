# Projekt: Minecraft-Klon

## Ziel
Entwickle ein funktionsfähiges, einfaches Minecraft-ähnliches Spiel. Der Spieler
soll sich in einer 3D-Welt aus Blöcken frei bewegen, Blöcke abbauen und platzieren
können. Das Spiel muss spielbar und stabil laufen.

## Kernfunktionen (Muss)
- **3D-Welt aus Blöcken**: Eine Welt, die aus würfelförmigen Blöcken besteht.
- **Bewegung**: Spieler kann laufen, springen und sich mit der Maus umsehen
  (First-Person-Perspektive, WASD + Maus).
- **Blöcke abbauen & platzieren**: Linksklick entfernt einen Block, Rechtsklick
  setzt einen neuen Block.
- **Mehrere Blocktypen**: Mindestens Gras, Erde, Stein und Holz, optisch unterscheidbar.
- **Einfaches Terrain**: Eine generierte Landschaft mit Boden und etwas Höhenvariation.
- **Kollision & Schwerkraft**: Der Spieler fällt und steht auf Blöcken, läuft nicht durch sie hindurch.

## Erweiterungen (Kann, optional)
- Einfaches Inventar / Blockauswahl
- Tag-Nacht-Wechsel
- Speichern & Laden der Welt
- Hintergrundmusik / Soundeffekte
- Endlos generierte Welt (Chunks)

## Technische Hinweise
- Freie Wahl der Technologie (z. B. Python + Ursina/Panda3D, JavaScript + Three.js,
  oder eine Game-Engine). Empfehlung: etwas Einsteigerfreundliches.
- Code sauber strukturieren und mit kurzen Kommentaren versehen.
- Eine kurze README mit Anleitung zum Starten des Spiels bereitstellen.

## Erfolgskriterium
Das Spiel startet ohne Fehler und man kann sich bewegen sowie Blöcke abbauen und
platzieren. Damit gilt das Grundziel als erreicht.
