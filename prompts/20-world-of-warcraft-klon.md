# Projekt: World-of-Warcraft-Klon

## Ziel
Entwickle ein funktionsfähiges, einfaches MMORPG im Stil von World of Warcraft.
Der Spieler steuert einen Helden durch eine 3D-Fantasy-Welt, bekämpft Gegner,
sammelt Beute und wird durch Erfahrung stärker. Das Spiel muss spielbar und
stabil laufen.

## Kernfunktionen (Muss)
- **3D-Fantasy-Welt**: Eine begehbare Spielwelt mit Landschaft, Wegen und Orten.
- **Spielercharakter**: Held in Third-Person-Perspektive, steuerbar mit WASD + Maus.
- **Kampfsystem**: Gegner angreifen, Schaden austeilen und einstecken; Gegner
  können besiegt werden.
- **Charakterwerte**: Lebenspunkte, Schaden und mindestens eine Ressource (z. B. Mana).
- **Stufenaufstieg**: Erfahrungspunkte durch besiegte Gegner, Level-Up macht den Helden stärker.
- **Fähigkeiten**: Mindestens 2–3 einsetzbare Fertigkeiten (z. B. Angriff, Heilung, Spezialangriff).
- **Gegner / Monster**: Computergesteuerte Feinde, die den Spieler erkennen und angreifen.

## Erweiterungen (Kann, optional)
- Quests / Aufgaben mit Belohnungen
- Inventar & Ausrüstung (Waffen, Rüstung)
- Verschiedene Klassen oder Rassen
- NPCs mit Dialogen und Händlern
- Mehrspieler / Online-Modus
- Karte & Minimap

## Technische Hinweise
- Freie Wahl der Technologie (z. B. Unity, Godot, oder JavaScript + Three.js).
  Empfehlung: eine Game-Engine, die 3D und Animationen unterstützt.
- Code sauber strukturieren und mit kurzen Kommentaren versehen.
- Eine kurze README mit Anleitung zum Starten des Spiels bereitstellen.

## Erfolgskriterium
Das Spiel startet ohne Fehler, man kann den Helden bewegen, Gegner bekämpfen und
durch Erfahrung aufsteigen. Damit gilt das Grundziel als erreicht.
